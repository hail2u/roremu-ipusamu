ロレム・イプサム
================

Web ページや Web アプリケーション等のモックアップに使うためのそこそこ意味の通った英語交じりの日本語の文章を登録・収集するプロジェクトです。


ルール
------

  - 自分で書いた文章であること
  - CC0 で公開することに同意すること
  - 公序良俗に反しない文章であること
  - 政治・宗教に関わりのない文章であること
  - フォーマットが [Markdown][1] であること
  - 見出しが 25 文字以下程度であること
  - 一段落が 150 文字から 300 文字程度であること
  - 本文に 1 割以上は英語（数字含む）が含まれること
  - GitHub のユーザー名に .md という拡張子をつけたファイル名にすること
